## üêõ Troubleshooting

### Trades losing?
- Check win rate after 20+ trades
- If <50%, stop and investigate
- Verify using correct timeframe (300 = 5M)

### Connection issues?
- Bot auto-reconnects
- Check `/status` for connection status
- Restart bot if needed: `/shutdown` then restart
